## 🔧 Frequently Asked Questions

If you encounter issues, look through these common resolutions:

The app will not open.
Restart your computer to ensure no other programs block the installation. Verify that your antivirus software did not move the file to quarantine.

The dashboard says "Connection Failed".
Verify that you have an open network port. Sometimes firewall settings prevent the program from starting its local web server. Check your Windows Firewall settings to allow Flowdesk.

Where do I save my data?
The system saves data inside the installation folder by default. Back up this folder regularly to prevent loss of your ticket history.
